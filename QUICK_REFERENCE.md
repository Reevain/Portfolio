## 🚨 Troubleshooting

**Node not found:**
Install from https://nodejs.org

**Port 3000 in use:**
```bash
npm start -- --port 3001
```

**Setup fails:**
```bash
rm -rf src public node_modules
node setup-portfolio.js
npm install
```
