## 🚀 How to Get Started

### Quick Start (5 minutes)

1. **Open Terminal/Command Prompt:**
   ```bash
   cd c:\Users\ASUS\Desktop\Portfolio
   ```

2. **Organize Files (First Time Only):**
   ```bash
   node organize-files.js
   ```

3. **Install Dependencies:**
   ```bash
   npm install
   ```

4. **Start Development Server:**
   ```bash
   npm start
   ```

5. **Open Browser:**
   Navigate to `http://localhost:3000`

## 📞 Support & Troubleshooting

### Common Issues & Solutions

**Issue:** npm install fails
- Solution: `npm cache clean --force` then `npm install`

**Issue:** Port 3000 in use
- Solution: `npm start -- --port 3001`

**Issue:** Files not in right location
- Solution: Run `node organize-files.js`

**Issue:** Styles not loading
- Solution: Clear browser cache (Ctrl+Shift+Del)
